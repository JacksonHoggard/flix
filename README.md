Flix is a modular, multi-feature Discord bot. It was created for use with the Movie Knight discord server and features an abundance of management tools.

## Setup
To set up an instance of the bot:
1. Clone or download the repository.
2. In the root folder of the repository, create a token.txt file
3. Copy and paste a bot token into the file for use with an intended bot.
4. Run the bot in a preferred IDE or IDEA.
